# PasswdAdmin

A simple password administrator. 
![Logo](img/passwdAdmin_logo.png)


## Some Information
- Title: `PasswdAdmin`
- Author: [@Adriansolier322](https://www.github.com/Adriansolier322)
- Version: `0.2.1`


### Installation

To install the project with git run:
```bash
  git clone https://github.com/Adriansolier322/PasswdAdmin.git
```
### Dependencies
The main dependence is python3 and Pycryptodomex, to install the last one run:
```bash
  pip install pycryptodomex
```
## How to run
To run it normally, execute the program `PasswdAdmin-start.bat` because it is a program that detects if you have all the necessary dependencies for the correct execution.
If you are sure that you have all the dependencies run `PasswdAdmin.py` directly
